## Installation

```html
<script src="https://js.recurly.com/v4/recurly.js"></script>
```

## Build
Development build server
```bash
make
```
Build to the file system
```bash
$ make build
```

If you are having issues with the build, try `make clean`.

## Test
```bash
$ make test
```
